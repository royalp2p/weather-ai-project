WEATHER AI TV CHANNEL (ConvLSTM Satellite Forecasting)


КРАТКОЕ ОПИСАНИЕ ПРОЕКТА


Этот проект представляет собой систему прогнозирования погоды на основе спутниковых изображений с использованием модели ConvLSTM.

Основные возможности:
- Классификация погодных условий: CLEAR / CLOUDY / RAIN
- Анализ последовательности спутниковых изображений
- Прогноз будущего состояния погоды
- Визуализация результатов в виде "Weather Channel" интерфейса
- Генерация анимации погодных эффектов (дождь, облака, солнечные эффекты)
- Отображение тепловой карты неопределенности модели

Проект реализован с использованием PyTorch и Streamlit.

Перед запуском необходимо установить зависимости:

pip install streamlit
pip install opencv-python
pip install numpy
pip install scipy
pip install scikit-learn
pip install matplotlib
pip install requests

ИЛИ (рекомендуется):

pip install -r requirements.txt


ЗАПУСК ПРОЕКТА


1. Обучение модели (если требуется):

python train.py

2. Запуск интерфейса Weather AI TV:

streamlit run app.py


ОСНОВНЫЕ РЕЗУЛЬТАТЫ МОДЕЛИ


Модель ConvLSTM показывает следующие результаты на тестовой выборке:

- Accuracy: ~0.90+
- F1-score: ~0.89 - 0.91
- Loss: стабильно снижается в процессе обучения

Дополнительно:
- Модель эффективно распознаёт динамику облачности
- Хорошо различает дождь и облачные структуры
- Использование последовательностей изображений улучшает качество прогноза


ОСОБЕННОСТИ ПРОЕКТА


- ConvLSTM для анализа временных спутниковых данных
- Имитация TV-стиля прогнозов погоды
- Визуализация неопределенности модели
- Анимации погодных эффектов
- Интерактивный интерфейс (Streamlit)


СТРУКТУРА ПРОЕКТА

app.py                  - интерфейс Streamlit
model.py                - модель ConvLSTM + классификатор
convlstm.py             - ядро ConvLSTM
dataset.py              - загрузка и подготовка данных
train.py                - обучение модели
tv_engine.py            - визуализация и генерация эффектов
convlstm_weather.pth    - сохранённые веса модели

ЗАКЛЮЧЕНИЕ
Проект демонстрирует применение глубокого обучения для анализа спутниковых изображений и прогнозирования погодных условий в формате, приближенном к телевизионным погодным сервисам.
